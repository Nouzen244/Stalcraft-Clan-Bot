# 🎮 StalZone Clan Bot

> Discord-бот для клана в игре **StalZone** (бывший STALCRAFT).
> Учёт посещаемости клановых войн (КВ) по этапам, система званий, календарь и выгрузка в Excel.
> ⚠️ Проект в активной разработке.

## ✨ Возможности

- **Фиксированное расписание КВ** — Потасовки, Турниры и Захват базы по дням недели (зашито в коде, не редактируется).
- **Этапы боёв** — присутствие учитывается по каждому этапу КВ.
- **Автоматический учёт** — кто был в голосовом канале на момент начала КВ или зашёл позже отмечается автоматически.
- **Звания** — фиксированная иерархия; звание берётся из роли Discord или назначается вручную.
- **Календарь КВ картинкой** (`!calendar`) с цветными метками событий и выбором воскресного боя.
- **Экспорт в Excel** — лист за день (с этапами) и матрица посещаемости по всем датам.
- **Детальный трекинг голосовых** — время входа/выхода, длительность каждой сессии.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```
(нужен Python 3.10+; для календаря-картинки используется Pillow)

### 2. Конфигурация
Скопируйте `config.example.json` в `config.json` и впишите токен бота:
```json
{
    "TOKEN": "ВАШ_ТОКЕН_БОТА",
    "PREFIX": "!",
    "TIMEZONE": "Europe/Moscow"
}
```
`config.json` в `.gitignore` — токен не попадёт в репозиторий.

### 3. Права бота
В [Discord Developer Portal](https://discord.com/developers/applications) → ваше приложение → **Bot** включите:
- ✅ SERVER MEMBERS INTENT
- ✅ MESSAGE CONTENT INTENT

### 4. Запуск
```bash
python main.py
```

### 5. Настройка на сервере
```
!setup                    — мастер настройки
!setvc #канал-кв          — голосовой канал КВ
!setchannel report #канал — канал уведомлений/отчётов
!setchannel log #канал    — канал логов входов/выходов
!setleader @Лидер         — привязка ролей к званиям
!setofficer @Офицер
!setprivate @Рядовой
```

## 📋 Команды

### ⚙️ Настройка (Администратор)
| Команда | Описание |
|---------|----------|
| `!setup` | Мастер настройки |
| `!setvc #канал` | Голосовой канал КВ |
| `!setchannel report #канал` | Канал отчётов/уведомлений |
| `!setchannel log #канал` | Канал логов |
| `!status` | Текущий статус бота |

### 🪖 Звания (Администратор)
Иерархия: **Лидер › Полковник › Офицер › Сержант › Боец › Рядовой**.
Звание = ручное назначение (`!setrank`) → роль Discord (`!setrole`) → Рядовой.

| Команда | Описание |
|---------|----------|
| `!setleader @роль` … `!setprivate @роль` | Привязать роль Discord к званию (по команде на звание) |
| `!setrole <звание> @роль` | То же одной универсальной командой |
| `!removerole <звание> @роль` | Убрать привязку роли |
| `!setrank @игрок <звание>` | Назначить звание вручную |
| `!delrank @игрок` | Убрать ручное звание |
| `!rank [@игрок]` | Показать звание |
| `!roles` | Иерархия и привязки |

### ⚔️ КВ (Клановые войны)
| Команда | Описание |
|---------|----------|
| `!schedule` | Фиксированное расписание КВ |
| `!calendar` | Календарь КВ на месяц (картинкой) |
| `!kv` | Кто сейчас в голосовом (во время КВ) — *Офицер+* |
| `!kv 17-01-2026` | Отчёт за конкретную дату — *Офицер+* |
| `!kvedit` | Редактировать посещаемость (присутствовал / отсутствовал / У/П) — *Офицер+* |

### 📊 Статистика и отчёты
| Команда | Описание |
|---------|----------|
| `!me [@игрок]` | Личная статистика по КВ (чужая — *Офицер+*) |
| `!stats @игрок` | То же, что `!me` |
| `!top10` | Топ-10 по посещаемости КВ |
| `!online` | Кто сейчас в голосовых каналах |
| `!export [дата]` | Выгрузка в Excel — *Офицер+* |

### ℹ️ Сервис
| Команда | Описание |
|---------|----------|
| `!help` | Справка |
| `!ping` | Проверка отклика |
| `!uptime` | Время работы бота |

## 🕐 Расписание КВ (фиксированное)

Расписание постоянное и **не редактируется** командами.

| Дни | Событие | Время | Этапы |
|-----|---------|-------|-------|
| Пн, Вт, Ср | 🔵 Потасовка | 20:00–21:00 | 20:00–20:20 · 20:20–20:40 · 20:40–21:00 |
| Чт, Пт, Сб | 🔴 Турнир | 20:00–21:15 | 20:00–20:25 · 20:25–20:50 · 20:50–21:15 |
| Вс | 🔵 Потасовка / 🟡 Захват базы | — | этапы Захвата базы — в разработке |

В **воскресенье** офицер выбирает событие — Потасовка или Захват базы — кнопками в `!kv` или `!calendar`. Кнопкой «Запомнить мой выбор» можно зафиксировать его для всех воскресений.

## 🪖 Иерархия званий

```
👑 Лидер
🎖️ Полковник
⚔️ Офицер        ← Офицер и выше = права управления ботом
🛡️ Сержант
🔫 Боец
🪖 Рядовой
```

Все участники сервера (кроме ботов) — участники клана. Звание определяется ручным назначением (`!setrank`), иначе по привязанной роли Discord (`!setrole`), иначе — Рядовой.

## 📑 Экспорт в Excel (`!export`)

- **Лист «Сегодня»** — посещаемость за день: статус, время входа/выхода, сессии, суммарное время и **присутствие по каждому этапу**.
- **Лист «Посещаемость»** — матрица «участники × все даты КВ»: 🟩 был · 🟥 не был · 🟨 У/П, плюс счётчик и процент явки. Звание показывается у каждого.

## 🗃️ Структура БД (SQLite)

| Таблица | Назначение |
|---------|-----------|
| `guild_settings` | Настройки сервера (канал КВ, каналы отчётов/логов, выбор по умолчанию для Вс) |
| `guild_roles` | Привязки ролей Discord к званиям |
| `member_ranks` | Ручные звания участников |
| `kv_attendance` | Посещаемость КВ (присутствие, У/П, время в ГС) |
| `kv_event_choice` | Выбор воскресного события по датам |
| `voice_sessions` | Сессии в голосовых (вход/выход/длительность) |
| `action_logs` | Лог действий |

## ⚠️ Важно

1. **Токен** — храните в секрете, не коммитьте `config.json`.
2. **Intents** — обязательно включите SERVER MEMBERS и MESSAGE CONTENT в Developer Portal.
3. **Права бота** — он должен видеть нужные каналы и роли.
4. **Timezone** — укажите свой часовой пояс в `config.json`.
