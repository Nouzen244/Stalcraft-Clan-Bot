# 🎮 StalZone Clan Bot

> Бот для клана в игре **StalZone** (бывший STALCRAFT).

Discord-бот для управления кланом с **динамическим управлением** ролями, расписанием и детальным трекингом.

## ✨ Новые возможности v2.0

### 🔧 Динамическое управление
- **Роли через команды** — админы назначают роли без редактирования конфига
- **Выбор VC через команду** — `!setvc #канал`
- **Расписание отслеживания** — бот следит только в указанное время

### 📊 Детальный трекинг
- Время входа в ГС
- Время выхода из ГС
- Продолжительность каждой сессии
- Количество входов за день
- Связь с расписанием (сессией)

## 🚀 Быстрый старт

### 1. Установка
```bash
pip install -r requirements.txt
```

### 2. Конфигурация
В `config.json` укажите только токен:
```json
{
    "TOKEN": "YOUR_BOT_TOKEN_HERE",
    "PREFIX": "!",
    "TIMEZONE": "Europe/Moscow"
}
```

### 3. Запуск
```bash
python main.py
```

### 4. Настройка на сервере
```
!setup              — мастер настройки
!setvc #канал       — установить VC для отслеживания
!setrole leader @Глава
!setrole officer @Офицер  
!setrole member @Клан
!schedule          — посмотреть расписание КВ (фиксированное)
```

## 📋 Команды

### ⚙️ Настройка (Администратор)
| Команда | Описание |
|---------|----------|
| `!setup` | Мастер начальной настройки |
| `!setvc #канал` | Установить VC для отслеживания |
| `!setchannel report #канал` | Канал для отчётов |
| `!setchannel log #канал` | Канал для логов |

### 🪖 Звания (Администратор)
Иерархия фиксированная: **Лидер › Полковник › Офицер › Сержант › Боец › Рядовой**.
Звание участника = ручное назначение в боте → роль Discord → Рядовой.

Для каждого звания — своя команда (привязка роли Discord):

| Команда | Звание |
|---------|--------|
| `!setleader @роль` | 👑 Лидер |
| `!setcolonel @роль` | 🎖️ Полковник |
| `!setofficer @роль` | ⚔️ Офицер |
| `!setsergeant @роль` | 🛡️ Сержант |
| `!setfighter @роль` | 🔫 Боец |
| `!setprivate @роль` | 🪖 Рядовой |

Прочие команды:

| Команда | Описание |
|---------|----------|
| `!setrole <звание> @роль` | То же одной универсальной командой |
| `!removerole <звание> @роль` | Убрать привязку роли |
| `!setrank @игрок <звание>` | Назначить звание вручную |
| `!delrank @игрок` | Убрать ручное звание |
| `!rank [@игрок]` | Показать звание |
| `!roles` | Иерархия и привязки |

### 🕐 Расписание КВ (фиксированное)
| Команда | Описание |
|---------|----------|
| `!schedule` | Показать расписание КВ (изменить нельзя) |

## 🕐 Расписание КВ (фиксированное)

Расписание постоянное и **не редактируется** командами. Все бои идут **20:00–21:00**:

| Дни | Событие |
|-----|---------|
| Пн, Вт, Ср | ⚔️ Потасовка |
| Чт, Пт, Сб | 🏆 Турнир |
| Вс | Потасовка / Захват базы (выбор) |

В **воскресенье** офицер выбирает событие — **Потасовка** или **Захват базы** — кнопками в отчёте `!kv`. Засчитывается одно событие.

Посмотреть расписание: `!schedule`.

## 🪖 Иерархия званий (фиксированная)

```
👑 Лидер
🎖️ Полковник
⚔️ Офицер
🛡️ Сержант      ← Офицер и выше = права управления
🔫 Боец
🪖 Рядовой
```

Другой иерархии быть не может. Звание берётся из ручного назначения
(`!setrank`) либо из привязанной роли Discord (`!setrole`), иначе — Рядовой.

## 📊 Что отслеживается

При каждом входе/выходе в VC записывается:

| Поле | Описание |
|------|----------|
| `join_time` | Точное время входа |
| `leave_time` | Точное время выхода |
| `duration_seconds` | Длительность в секундах |
| `date` | Дата (YYYY-MM-DD) |
| `user_id` | Discord ID |
| `display_name` | Ник в Discord |
| `channel_name` | Название канала |
| `schedule_id` | ID сессии (если по расписанию) |
| `status` | active / completed / interrupted |

## 🗃️ Структура БД

### guild_settings
Настройки каждого сервера (VC, каналы).

### guild_roles  
Привязки ролей Discord к званиям (leader, colonel, officer, sergeant, fighter, private).

### tracking_schedule
Расписание отслеживания (время, дни недели).

### voice_sessions
Детальные сессии (вход, выход, длительность).

### attendance
Суммарная посещаемость за день.

!export
