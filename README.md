# 🎮 STALCRAFT Clan Bot v2.0

Discord-бот для управления кланом с **динамическим управлением** ролями, расписанием и детальным трекингом.

## ✨ Новые возможности v2.0

### 🔧 Динамическое управление
- **Роли через команды** — админы назначают роли без редактирования конфига
- **Выбор VC через команду** — `!setvc #канал`
- **Расписание отслеживания** — бот следит только в указанное время

### 📊 Детальный трекинг
- Время входа в ГС
- Время выхода из ГС
- Продолжительность каждой сессии
- Количество входов за день
- Связь с расписанием (сессией)

## 🚀 Быстрый старт

### 1. Установка
```bash
pip install -r requirements.txt
```

### 2. Конфигурация
В `config.json` укажите только токен:
```json
{
    "TOKEN": "YOUR_BOT_TOKEN_HERE",
    "PREFIX": "!",
    "TIMEZONE": "Europe/Moscow"
}
```

### 3. Запуск
```bash
python main.py
```

### 4. Настройка на сервере
```
!setup              — мастер настройки
!setvc #канал       — установить VC для отслеживания
!setrole leader @Глава
!setrole officer @Офицер  
!setrole member @Клан
!schedule add 20:00-21:30 Вечерний сбор
```

## 📋 Команды

### ⚙️ Настройка (Администратор)
| Команда | Описание |
|---------|----------|
| `!setup` | Мастер начальной настройки |
| `!setvc #канал` | Установить VC для отслеживания |
| `!setchannel report #канал` | Канал для отчётов |
| `!setchannel log #канал` | Канал для логов |

### 👑 Управление ролями (Администратор)
| Команда | Описание |
|---------|----------|
| `!setrole leader @роль` | Назначить роль Главы |
| `!setrole officer @роль` | Назначить роль Офицера |
| `!setrole member @роль` | Назначить роль Рядового |
| `!setrole special @роль` | Назначить особую роль |
| `!setrole recruit @роль` | Назначить роль Рекрута |
| `!roles` | Показать все роли |
| `!removerole <тип> @роль` | Удалить роль |

### 🕐 Расписание (Офицер+)
| Команда | Описание |
|---------|----------|
| `!schedule add 20:00-21:30 Название` | Добавить слот |
| `!schedule add 12:00-13:00 Сбор пн,ср,пт` | С днями недели |
| `!schedule list` | Показать расписание |
| `!schedule remove <ID>` | Удалить слот |
| `!schedule clear` | Очистить всё |

### 📊 Посещаемость (Офицер+)
| Команда | Описание |
|---------|----------|
| `!report` | Отчёт о присутствии |
| `!stats @user` | Статистика игрока |
| `!top10` | Топ-10 активных |
| `!online` | Кто сейчас в VC |
| `!sessions @user [дата]` | Детальные сессии |
| `!export [дата]` | Экспорт в Excel |

### ℹ️ Информация
| Команда | Описание |
|---------|----------|
| `!status` | Статус отслеживания |
| `!ping` | Проверка бота |
| `!uptime` | Время работы |
| `!help` | Справка |

## 🕐 Расписание отслеживания

Бот может отслеживать VC только в определённое время:

```
# Добавить вечерний сбор (каждый день)
!schedule add 20:00-21:30 Вечерний сбор

# Добавить дневной сбор (только Пн, Ср, Пт)
!schedule add 12:00-13:00 Дневной сбор пн,ср,пт

# Несколько слотов в день
!schedule add 12:00-13:00 Утренний
!schedule add 20:00-22:00 Вечерний
```

**Если расписание не задано** — бот отслеживает 24/7.

## 👥 Иерархия ролей

```
👑 Leader (Глава)
   ↓ Полные права
⚔️ Officer (Офицер)  
   ↓ Отчёты, обработка, расписание
⭐ Special (Особая)
   ↓ Настраиваемые права
🎖️ Member (Рядовой)
   ↓ Отслеживание, своя статистика
🆕 Recruit (Рекрут)
   ↓ Базовое отслеживание
```

## 📊 Что отслеживается

При каждом входе/выходе в VC записывается:

| Поле | Описание |
|------|----------|
| `join_time` | Точное время входа |
| `leave_time` | Точное время выхода |
| `duration_seconds` | Длительность в секундах |
| `date` | Дата (YYYY-MM-DD) |
| `user_id` | Discord ID |
| `display_name` | Ник в Discord |
| `channel_name` | Название канала |
| `schedule_id` | ID сессии (если по расписанию) |
| `status` | active / completed / interrupted |

## 🗃️ Структура БД

### guild_settings
Настройки каждого сервера (VC, каналы).

### guild_roles  
Динамические роли (leader, officer, member, special, recruit).

### tracking_schedule
Расписание отслеживания (время, дни недели).

### voice_sessions
Детальные сессии (вход, выход, длительность).

### attendance
Суммарная посещаемость за день.

## 🔧 Права бота

- ✅ SERVER MEMBERS INTENT
- ✅ MESSAGE CONTENT INTENT

## 📝 Пример использования

```
# 1. Первичная настройка
!setvc #голосовой-канал
!setrole leader @Глава-клана
!setrole officer @Офицер
!setrole member @Клановец

# 2. Настройка расписания
!schedule add 20:00-21:30 Вечерний рейд
!schedule add 12:00-13:00 Дневной сбор пн,ср,пт

# 3. Проверка статуса
!status

# 4. Генерация отчёта (во время сессии)
!report

# 5. Просмотр детальных сессий
!sessions @Игрок 2024-01-15

# 6. Экспорт в Excel
!export
```

## ⚠️ Важно

1. **Токен** — храните в секрете!
2. **Intents** — обязательно включите в Developer Portal
3. **Права** — бот должен видеть каналы и роли
4. **Timezone** — укажите вашу временную зону в config.json
